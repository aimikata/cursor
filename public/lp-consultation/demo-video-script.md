# デモ動画スクリプト
## Kindle印税資産 個別相談LP用（3〜5分）

---

## 動画の目的

1. **「プロジェクト作成」「世界観設計」は簡単そう** → 興味を持たせる
2. **「でも1冊作るのは大変そう」** → フィルター効果（真剣な人だけが相談に来る）
3. **「個別相談で詳しく説明します」** → 相談申込へ誘導

---

## 構成（3〜5分）

| 時間 | シーン | 内容 |
|------|--------|------|
| 0:00-0:15 | オープニング | タイトル・挨拶 |
| 0:15-1:00 | プロジェクト作成 | 簡単そうに見せる |

| 1:45-2:30 | ストーリー生成 | 簡単そうに見せる |

## 詳細スクリプト

### 【0:00-0:15】オープニング

**画面：**
- タイトル「90分で1冊のマンガを出版する仕組み」
- サブタイトル「実際の操作画面をご紹介します」

**ナレーション：**
```
「こんにちは。Kindle印税資産の個別相談にお申し込みいただき、ありがとうございます。
今日は、90分で1冊のマンガを出版する仕組みを、実際の操作画面でご紹介します。」
```

---

### 【0:15-1:00】プロジェクト作成（簡単そうに見せる）

**画面操作：**
- ツールを開く
- 「新規プロジェクト作成」をクリック
- プロジェクト名を入力（例：「投資初心者向けマンガ」）
- 「作成」ボタンをクリック
- 数秒でプロジェクトが作成される

**ナレーション：**
```
「まず、プロジェクトを作成します。
プロジェクト名を入力して、作成ボタンを押すだけ。
これで、新しいマンガ制作プロジェクトの準備が整いました。
操作は非常にシンプルです。」
```

**ポイント：**
- 操作が簡単そうに見える
- 迷うことなく進められる

---

### 【1:00-1:45】世界観設計（簡単そうに見せる）

**画面操作：**
- 「世界観設計」タブを開く
- テーマを選択（例：「投資・資産形成」）
- 「自動生成」ボタンをクリック
- AIが世界観を提案（数秒で表示）
- 提案された内容を確認

**ナレーション：**
```
「次に、世界観を設計します。
テーマを選んで、自動生成ボタンを押すだけ。
AIが世界観を提案してくれます。
例えば、『投資初心者が主人公の世界』『資産形成をテーマにした世界』など。
これも、ほぼ自動で完了します。」
```

**ポイント：**
- AIが自動で提案してくれる
- 選択するだけなので簡単

---

### 【1:45-2:30】ストーリー生成（簡単そうに見せる）

**画面操作：**
- 「ストーリー生成」タブを開く
- 世界観を選択
- 「ストーリー生成」ボタンをクリック
- AIがストーリーを生成（数秒で表示）
- 生成されたストーリーをスクロールで確認

**ナレーション：**
```
「次に、ストーリーを生成します。
先ほど作成した世界観を選んで、ストーリー生成ボタンを押すだけ。
AIが自動でストーリーを作成してくれます。
文章力は一切必要ありません。
数秒で、マンガのストーリーが完成します。」
```

**ポイント：**
- 文章力不要
- ボタン1つで完成

---

### 【2:30-3:00】コマ割り・画像生成（「でも大変そう」）

**画面操作：**
- 「コマ割り・画像生成」タブを開く
- ストーリーを選択
- 「コマ割り生成」ボタンをクリック
- コマ割りが表示される（複数のコマ）
- 「画像生成」ボタンをクリック
- 画像生成が開始される（進行中の画面）
- 「生成中... 残り時間：約25分」のような表示

**ナレーション：**
```
「ここからが、少し時間がかかります。
ストーリーに基づいて、コマ割りと画像を生成します。
コマ割りは自動で作成されますが、
画像生成には時間がかかります。
1冊分の画像を生成するのに、約25分かかります。
これが、1冊を作る上で最も時間がかかる工程です。」
```

**ポイント：**
- 「25分かかる」と強調
- 「大変そう」という印象を与える
- でも、実際には待つだけなので簡単

---

### 【3:00-3:30】クロージング（個別相談へ誘導）

**画面：**
- 完成イメージ（表紙と数ページ）
- 「90分で1冊完成」の文字

**ナレーション：**
```
「プロジェクト作成から世界観設計、ストーリー生成まで、
これらは非常に簡単にできます。
でも、1冊のマンガを完成させるには、
画像生成やその後の工程も必要です。

実は、この仕組みを使えば、全体で90分で1冊完成します。
なぜ90分でできるのか？
その理由と、具体的な手順を、
個別相談で詳しくお伝えします。

あなた専用の印税資産シミュレーションも作成しますので、
ぜひ個別相談にお申し込みください。」
```

**画面：**
- 「個別相談に申し込む（1,100円）」ボタン
- 「24時間限定価格」の表示

## 撮影・編集のポイント

### 画面録画のコツ
1. **操作はゆっくり**（視聴者が追いやすい速度）
2. **カーソルを大きく**（Windowsの設定でカーソルサイズを大きく）
3. **不要な部分はカット**（待ち時間など）
4. **重要な部分はスローモーション**（ボタンクリックなど）

### ナレーションのコツ
1. **ゆっくり話す**（視聴者が理解できる速度）
2. **重要な部分は強調**（「90分」「自動」「簡単」など）
3. **「でも大変そう」の部分は少し重く**（フィルター効果）

### 編集のコツ
1. **BGMは控えめに**（操作音やナレーションが聞こえるように）
2. **テロップを追加**（「プロジェクト作成」「世界観設計」など）
3. **時間表示**（「0:30 プロジェクト作成」など）
