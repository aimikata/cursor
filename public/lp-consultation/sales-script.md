# セールストークスクリプト
## Kindle印税資産 無料相談 → 30万円成約

---

## 全体の流れ（30〜45分）

| 時間 | フェーズ | 目的 |
|------|----------|------|
| 0〜5分 | アイスブレイク | 緊張をほぐす、関係構築 |
| 5〜15分 | ヒアリング | 課題・目標を明確化 |
| 15〜25分 | シミュレーション | 具体的な数字で納得させる |
| 25〜35分 | 仕組みの説明 | 再現性を証明 |
| 35〜40分 | 提案・価格提示 | オファーを出す |
| 40〜45分 | クロージング | 決断を促す |

---

## フェーズ1：アイスブレイク（0〜5分）

### 目的
- 緊張をほぐす
- 話しやすい雰囲気を作る

### トーク例

```
「本日はお申し込みいただきありがとうございます。
○○と申します。よろしくお願いします。」

「今日は○○さん専用の印税資産シミュレーションを
作成させていただきますので、リラックスしてお話しください。」

「まず、簡単に自己紹介させていただきますね。
（簡潔に自己紹介：1分以内）」

「では、○○さんのことを教えていただければと思います。」
```

---

## フェーズ2：ヒアリング（5〜15分）

### 目的
- 現状の課題を明確にする
- 目標を具体化する
- 「この人は私のことを理解してくれている」と感じさせる

### 質問リスト

#### 1. きっかけ
```
「今回、無料相談にお申し込みいただいたきっかけは何ですか？」

「Kindle印税という言葉は、どこで知りましたか？」
```

#### 2. 現状の課題
```
「今のお仕事や収入について、何か課題を感じていますか？」

「労働収入以外の収入源について、今までに何か試されたことはありますか？」
（例：投資、副業、不動産など）

「うまくいかなかった理由は、何だと思いますか？」
```

#### 3. 目標
```
「もし印税収入が入るようになったら、どのくらいを目指したいですか？
月5万円くらい？10万円くらい？それ以上？」

「その収入があったら、生活はどう変わりそうですか？」
```

#### 4. 時間
```
「週にどのくらいの時間を使えそうですか？」

「本業との両立は問題なさそうですか？」
```

### ヒアリングのポイント

- **聞き役に徹する**（8割聞く、2割話す）
- **共感を示す**（「なるほど」「分かります」）
- **メモを取る**（シミュレーションで使う）

## フェーズ3：シミュレーション提示（15〜25分）

### 目的
- 具体的な数字で「できそう」と思わせる
- 目標達成までの道筋を示す

### トーク例

```
「ありがとうございます。○○さんのお話を聞いて、
シミュレーションを作成しますね。」

「○○さんの目標は月◯万円の印税収入でしたね。」

「これを達成するために必要な冊数を計算してみます。」
```

### シミュレーション例

```
【前提条件】
・1冊あたりの平均印税：月5,000円
 ※100冊以上出版しているコミュニティの実績平均
 ※Kindle Unlimitedのページ単価（0.4〜0.5円）から計算しても妥当な数字

・目標：月5万円の印税収入

【計算】
・必要冊数：5万円 ÷ 5,000円 = 10冊

【達成までの期間】
・1冊90分で作成可能
・週3冊ペースなら月12冊
・10冊達成まで：約1ヶ月

【その後】
・10冊達成後は、毎月5万円の印税が入り続ける
・20冊で月10万円、50冊で月25万円、100冊で月50万円も可能
```

### 根拠の説明（質問された場合）

```
「なぜ1冊5,000円なのか、根拠をお伝えしますね。」

「Kindle Unlimitedのページ単価は、1ページあたり約0.4〜0.5円です。」

「例えば、100ページの本が月に100回読まれた場合、
100ページ × 0.45円 × 100回 = 約4,500円になります。」

「150ページの本なら、月80回読まれれば5,400円。
これが1冊5,000円という数字の根拠です。」

「もちろん、売れる本・売れない本の差はあります。
でも、100冊以上出版している方々の平均が5,000円なので、
積み上げれば、この数字に近づいていきます。」
```

### トーク例（シミュレーション後）

```
「このように、○○さんの場合は約◯ヶ月で
月◯万円の印税収入が見込めます。」

「もちろん、1冊あたりの印税は本によって変動しますが、
"積み上がる"という構造は変わりません。」

「1冊作れば、それが来月も再来月も印税を生み続ける。
これが"資産"としてのKindle出版です。」

「ここまでで、何かご質問はありますか？」
```

## フェーズ4：仕組みの説明（25〜35分）

### 目的
- 「自分でもできそう」と思わせる
- 再現性を証明する

### トーク例

```
「では、どうやって90分で1冊出版できるのか、
具体的な仕組みをお見せします。」
```

### 仕組みの説明

```
【ステップ1：リサーチ（3分）】
「まず、売れるテーマを見つけます。
専用ツールがAmazonの売れ筋を分析して、
"勝ち筋"のテーマを提案してくれます。」

【ステップ2：世界観・ストーリー設計（8分）】
「次に、本の世界観とストーリーを設計します。
これもAIが自動で生成してくれるので、
文章力は必要ありません。」

【ステップ3：コマ割り・画像生成（30分）】
「ストーリーに基づいて、コマ割りと画像を生成します。
絵心がなくても、AIが自動で作成します。」

【ステップ4：校正（3分）】
「誤字脱字をAIがチェックします。」

【ステップ5：出版アシスト（3分）】
「Amazonに最適化されたキーワードやカテゴリを
自動で提案してくれます。」

【合計：約90分】
「これで1冊の出版準備が完了します。」
```

### ポイント

```
「この仕組みのポイントは、
"スキルに依存しない"ということです。」

「文章力も、絵心も、デザインセンスも必要ありません。
仕組みに従って進めるだけで、出版できます。」

「だから、再現性がある。
誰がやっても同じ結果が出るんです。」
```

## フェーズ5：提案・価格提示（35〜40分）

### 目的
- オファーを明確に伝える
- 価格に対する価値を感じさせる

### トーク例

```
「○○さん、ここまでの話を聞いて、いかがですか？」

（反応を確認）

「もし興味があれば、この仕組み一式をお渡しできます。」
```

### オファー内容

```
【提供内容】
1. 出版ツール一式（10種類以上）
   - リサーチツール
   - 世界観・ストーリー生成ツール
   - コマ割り・画像生成ツール
   - 校正ツール
   - 出版アシスタント
   - 表紙生成ツール
   など

2. 1冊出版までの1対1フルサポート
   - 初回セットアップサポート
   - 1冊目の出版まで伴走
   - 質問し放題

3. マニュアル一式

【価格】
30万円（税込）
```

### トーク例（価格提示）

```
「価格は30万円です。」

（間を置く）

「高いと感じるかもしれません。
でも、考えてみてください。」

「この30万円で、毎月印税が入る"資産"を
積み上げる仕組みが手に入ります。」

「1冊500円の印税として、100冊で月5万円。
年間60万円の印税収入です。」

「つまり、半年で元が取れる計算です。
その後は、ずっとプラスになります。」

「これは"コスト"ではなく"投資"です。」
```

## フェーズ6：クロージング（40〜45分）

### 目的
- 決断を促す
- 不安を解消する

### パターン1：前向きな反応の場合

```
「○○さん、やってみたいと思いますか？」

（「はい」の場合）

「ありがとうございます。
では、お申し込みの手続きを進めさせていただきますね。」
```

### パターン2：迷っている場合

```
「何か気になる点はありますか？」

（不安を聞き出す）

「なるほど。それは○○ということですね。」

（不安を解消する回答をする）
```

### よくある不安と回答

#### 「本当に自分でもできますか？」
```
「1冊目は私がフルサポートします。
一緒に作業しながら進めるので、
やり方が分からなくても大丈夫です。

2冊目以降は、1冊目の経験があるので
自分で進められるようになります。」
```

#### 「30万円は高いです」
```
「分かります。決して安い金額ではありません。

でも、考えてみてください。
これは"使って終わり"のものではなく、
"毎月収入を生む資産"を作る仕組みです。

月5万円の印税が入るようになれば、
半年で元が取れます。
その後は、ずっとプラスです。

"コスト"ではなく"投資"として
考えていただければと思います。」
```

#### 「もう少し考えたいです」
```
「もちろん、大きな決断ですからね。

ただ、一つお伝えしたいことがあります。」

（ここで隠し玉を出す → 下記参照）
```

## 隠し玉：英語圏市場（迷っている場合のみ）

### タイミング
- 価格で迷っている時
- 「考えたい」と言われた時
- 最後の一押しが必要な時

### トーク例

```
「実は、このツールにはもう一つ、
まだお伝えしていない強みがあります。」

「このツールは、英語圏のKindle市場にも
出版できるんです。」

「○○さん、英語は話せますか？」

（たいてい「いいえ」と答える）

「大丈夫です。英語が分からなくてもOKです。
校正ツールが英語の誤字脱字もチェックしてくれます。」

「ご存知ですか？
英語圏のKindleユーザーは、日本の10倍います。
でも、漫画家の数は日本の10分の1以下。」

「つまり、競合がほとんどいないんです。」

「"まんがでわかる"系のハウツー本は、
英語圏にはほとんど存在しません。
日本では当たり前のジャンルが、向こうでは"新しい"んです。」

「これは、購入された方にだけお伝えしています。
先行者利益を守るためなので、
SNS等での発信はお控えいただいています。」

「日本市場だけでなく、英語圏10億人市場も狙える。
これが、この仕組みの本当の価値です。」
```

## クロージングフレーズ集

### 決断を促す

```
「○○さん、始めてみませんか？」

「一緒に1冊目を出版しましょう。」

「迷っている時間も、資産は積み上がりません。
今日から始めれば、◯ヶ月後には最初の印税が入ります。」
```

### 今日決める理由

```
「今日お申し込みいただければ、
今週中に初回セットアップを行えます。」

「来週から、実際に1冊目の作成を始められます。」
```

### 最終確認

```
「では、お申し込みということでよろしいですか？」

（「はい」の場合）

「ありがとうございます。
では、決済の手続きをご案内しますね。」
```

---

## NGワード・NG行動

| NG | 理由 |
|----|------|
| 「絶対に稼げます」 | 収入保証はできない |
| 「誰でも簡単に」 | 誇大表現 |
| 「今日だけ特別価格」 | 嘘になる可能性 |
| 相手の話を遮る | 信頼を失う |
| 強引に押し込む | 後でキャンセルになる |

## 成約後の流れ

```
1. 決済手続き（30万円）
2. 決済確認後、ツール一式をお渡し
3. 初回セットアップ日程を調整
4. 初回セットアップ（Zoom）
5. 1冊目の出版まで伴走サポート
```
