## スライド1：オープニング（0:00-0:15）

### プロンプト
```

## スライド2：プロジェクト作成（0:15-1:00）

### プロンプト
```

【右側】

  2. 作成ボタンをクリック
  3. 数秒で完了」

【スタイル】

- 操作画面が主役
```

### ナレーション原稿
```
「まず、プロジェクトを作成します。
プロジェクト名を入力して、作成ボタンを押すだけ。
これで、新しいマンガ制作プロジェクトの準備が整いました。
操作は非常にシンプルです。」

## Gemini 3.0 プロンプトの使い方

### 各スライドの生成方法

1. **上記のプロンプトをGemini 3.0に入力**
2. **インフォグラフィック画像を添付**
3. **「ツールの操作画面のスクリーンショット」も添付**（可能であれば）
4. **生成されたスライドを確認・調整**

### 注意点

- **インフォグラフィックは背景として薄く配置**
- **操作画面が主役になるように**
- **テキストは読みやすく、金色（#d4af37）を使用**
- **各スライドは3〜5秒程度の表示時間**

---

## 動画編集の流れ

1. **スライド生成**（Gemini 3.0）
2. **スライドを動画に変換**（PowerPoint / Canva）
3. **AI音声生成**（ElevenLabs等）
4. **動画編集**（DaVinci Resolve / CapCut）
   - スライド + AI音声 + BGM（控えめに）
