# デモ動画 ナレーション原稿
## AI音声生成用（3〜5分）

---

## スライド1：オープニング（0:00-0:15）

```
こんにちは。Kindle印税資産に興味を持っていただき、ありがとうございます。
今日は、90分で1冊のマンガを出版する仕組みを、実際の操作画面でご紹介します。
```

---

## スライド2：プロジェクト作成（0:15-1:00）

```
まず、プロジェクトを作成します。
プロジェクト名を入力して、作成ボタンを押すだけ。
これで、新しいマンガ制作プロジェクトの準備が整いました。
操作は非常にシンプルです。
```

---

## スライド3：世界観設計（1:00-1:45）

```
次に、世界観を設計します。
テーマを選んで、自動生成ボタンを押すだけ。
AIが世界観を提案してくれます。
例えば、『投資初心者が主人公の世界』『資産形成をテーマにした世界』など。
これも、ほぼ自動で完了します。
```

---

## スライド4：ストーリー生成（1:45-2:30）

```
次に、ストーリーを生成します。
先ほど作成した世界観を選んで、ストーリー生成ボタンを押すだけ。
AIが自動でストーリーを作成してくれます。
文章力は一切必要ありません。
数秒で、マンガのストーリーが完成します。
```

---

## スライド5：コマ割り（2:30-2:50）

```
次に、ストーリー原稿をコマ割りに変換します。
原稿をアップロードすると、AIが自動でページ構造を解析し、
最適なコマ割りテンプレートを選択します。
38種類のテンプレートから、ストーリーに最適なものが選ばれます。
```

---

## スライド6：画像生成（2:50-3:15）

```
ここからが、少し時間がかかります。
テンプレートとキャラ画像、シーン指示を組み合わせて、
AIが画像を生成します。
1ページあたり10秒で生成されますが、
1冊分の画像を生成するのに、約25分かかります。
これが、1冊を作る上で最も時間がかかる工程です。
```

---

## スライド7：校正（3:15-3:30）

```
完成したマンガ画像をアップロードすると、
AIが自動で誤字脱字をチェックします。
10ページを10秒でチェックできるので、
校正も非常に簡単です。
```

---

## スライド8：出版設定（3:30-3:45）

```
最後に、Kindle出版に必要な設定を行います。
完成した作品をアップロードして、
お気に入りの3ページを選ぶだけ。
AIが自動で、キーワードや紹介文、A+コンテンツ用の文章を生成します。
これも、ほぼ自動で完了します。
```

---

## スライド9：完成イメージ（3:45-4:00）

```
プロジェクト作成から世界観設計、ストーリー生成まで、
これらは非常に簡単にできます。
でも、1冊のマンガを完成させるには、
画像生成やその後の工程も必要です。

実は、この仕組みを使えば、全体で90分で1冊完成します。
```

---

## スライド10：クロージング（4:00-4:30）

```
なぜ90分でできるのか？
その理由と、具体的な手順を、
個別相談で詳しくお伝えします。

あなた専用の印税資産シミュレーションも作成しますので、
ぜひ個別相談にお申し込みください。
```

---

## AI音声生成時の注意点

### 読み上げ速度
- **ゆっくり目**（視聴者が理解できる速度）
- 各スライドの表示時間に合わせて調整

### 強調ポイント
- 「90分」「自動」「簡単」などの重要キーワードは少し強めに
- 「でも25分かかります」の部分は少し重く（フィルター効果）

### トーン
- **プロフェッショナルで親しみやすい**
- 過度に興奮しない
- 落ち着いた説明口調
