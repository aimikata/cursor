# YouTubeチャンネル設定「manga-sensei」

チャンネル名 **manga-sensei** の設定用に、コピペできる文とチェックリストをまとめました。  
6本すべてアップ済みの前提で、チャンネル全体の見せ方を整えるためのものです。

---

## 設定の流れ（どこで何をするか）

| 順番 | 項目 | 設定場所 | コピペ用 |
|------|------|----------|----------|
| 1 | チャンネル名・説明・リンク | **YouTube Studio** → カスタマイズ → **基本情報** | 下記「基本情報」 |
| 2 | プロフィール写真・バナー | **YouTube Studio** → カスタマイズ → **ブランディング** | 下記「ブランディング」 |
| 3 | チャンネル上の表示順・セクション | **YouTube Studio** → カスタマイズ → **レイアウト** | 下記「レイアウト」 |

---

## 1. 基本情報（チャンネル説明・リンク）

**開き方**：YouTube Studio → 左メニュー「カスタマイズ」→「基本情報」

### チャンネル名

```
manga-sensei
```

（すでに「manga-sensei」になっていればそのままでOK）

---

### 説明文（チャンネルの「概要」欄に貼る）

```
Kindle印税を「仕組み」で資産にする方法を、マンガ制作ツールと数字で解説しています。

・90分で1冊のマンガを出版する仕組み
・印税収入を積み上げる考え方（不動産・株との比較）
・あなた専用の印税資産シミュレーションの相談

動画で仕組みのからくりを解説したあと、個別相談（有料）でシミュレーションを作成しています。
まずはリンク先のLPから、短い動画で「仕組み」を確認してみてください。

#Kindle出版 #印税 #資産形成 #電子書籍 #副業 #マンガ出版
```

（長さは問題なければこのまま。短くする場合は1〜3行目だけでも可）

---

### 連絡先メール（任意）

- ビジネス問い合わせ用のメールアドレスがあれば入力。
- なければ未設定のままでよい。

---

### ウェブサイト（必須推奨）

**表示されるリンク**の1つ目に、LPを出します。

| 項目 | 入力する値 |
|------|------------|
| **リンクのタイトル** | `無料で印税資産シミュレーションを作成する` または `個別相談はこちら` |
| **URL** | `https://lp-consultation.vercel.app/` |

2つ目以降は、X（Twitter）や公式サイトなどがあれば追加。

---

### ソーシャルリンク（任意）

- X（Twitter）、Instagram、Facebook など、公開したいURLを追加。

---

## 2. ブランディング（アイコン・バナー）

**開き方**：YouTube Studio → カスタマイズ → **ブランディング**

### プロフィール写真（チャンネルアイコン）

| 項目 | 内容 |
|------|------|
| **推奨サイズ** | 98×98px（表示は円形） |
| **実用的な作り方** | 800×800px の正方形画像を用意してアップロード（自動で縮小されます） |
| **中身の例** | 「manga-sensei」のロゴ、または顔写真・イラストの正方形トリミング |

---

### バナー画像（チャンネルアート）

| 項目 | 内容 |
|------|------|
| **推奨サイズ** | 2,560×1,440px（アスペクト比 16:9） |
| **安全エリア** | 中央 1,546×423px に重要情報を入れる（端はスマホで切れることがあります） |
| **文字・内容の例** | 「manga-sensei」「Kindle印税を資産に」「90分で1冊・印税の仕組み」＋ LPのURL または「個別相談▶ lp-consultation.vercel.app」 |

バナー用のデザインテキスト例（画像に載せる場合）：

```
manga-sensei
Kindle印税を「仕組み」で資産に
個別相談はこちら → lp-consultation.vercel.app
```

## 3. レイアウト（トップ画面の見せ方）

**開き方**：YouTube Studio → カスタマイズ → **レイアウト**

### おすすめの構成

| セクション | 内容の例 |
|------------|----------|
| **映像をはじめる**（一番上） | 本編動画（pCM2Dg3_iIk）を「おすすめ」として表示。LP誘導の軸にしたい動画を1本選ぶ。 |
| **ショート** | 30-1〜30-5 のショートが自動で並ぶ。順番は「追加した日」や人気順になるので、気になるものだけ並び替え可能。 |
| **投稿」タブ** | 本編・ショートが時系列で並ぶ。そのままでOK。 |

### チャンネル登録ボタンの上（「ウォーターマーク」以外）

- 「カスタムリンク」で **LP（https://lp-consultation.vercel.app/）** を表示しておくと、チャンネルページから直接相談へ誘導しやすいです。  
- 基本情報の「ウェブサイト」に同じURLを入れていれば、ここにも同じリンクを出せます。
