# セールストークスクリプト（完全版）
## 素人でも使える・売込み感ゼロ・顧客主導型

---

## 📋 基本原則

### セールスの心構え
1. **聞く：話す = 8：2**（顧客の話を8割聞く）
2. **押し付けない**（「買ってください」は絶対に言わない）
3. **共感を最優先**（「なるほど」「分かります」を多用）
4. **顧客が決断する**（自分から「これが欲しい」と言わせる）
5. **疑問をすべて解消**（不安を残さない）

### 絶対に言ってはいけない言葉
- ❌ 「絶対に稼げます」
- ❌ 「今だけ特別価格です」
- ❌ 「買ってください」
- ❌ 「決断してください」
- ❌ 「他の人も買っています」

### 使うべき言葉
- ✅ 「いかがですか？」
- ✅ 「どう思われますか？」
- ✅ 「気になる点はありますか？」
- ✅ 「他にご質問はありますか？」
- ✅ 「ご検討いただけますか？」

| 時間 | フェーズ | スライド | 目的 |
|------|----------|----------|------|
| 0〜5分 | アイスブレイク | - | 緊張をほぐす、関係構築 |
| 5〜10分 | ヒアリング | - | 課題・目標を明確化 |

| 40〜45分 | クロージング | - | 顧客の決断を待つ |

## 📞 フェーズ1：アイスブレイク（0〜5分）

### 目的
- 緊張をほぐす
- 話しやすい雰囲気を作る
- 信頼関係を構築

### トーク例

```
「本日はお時間をいただき、ありがとうございます。
○○と申します。よろしくお願いします。」

（少し間を置く）

「今日は、○○さん専用の印税資産シミュレーションを
一緒に作成させていただきます。
リラックスしてお話しくださいね。」

「まず、簡単に自己紹介させていただきますね。
（簡潔に自己紹介：1分以内）」

「では、○○さんのことを教えていただければと思います。
今日はどんなことを知りたいですか？」
```

### ポイント
- **顧客のペースに合わせる**（急かさない）
- **笑顔を忘れない**（声のトーンも明るく）
- **「知りたいですか？」と聞く**（押し付けない）

## 🎤 フェーズ2：ヒアリング（5〜10分）

### 目的
- 現状の課題を明確にする
- 目標を具体化する
- 「この人は私のことを理解してくれている」と感じさせる

### 質問の流れ

#### 1. きっかけを聞く（オープンクエスチョン）

```
「今回、無料相談にお申し込みいただいたきっかけは何ですか？」

（答えを待つ。遮らない）

「なるほど。Kindle印税という言葉は、どこで知りましたか？」
```

#### 2. 現状の課題を聞く

```
「今のお仕事や収入について、何か課題を感じていますか？」

（答えを待つ）

「なるほど。労働収入以外の収入源について、
今までに何か試されたことはありますか？」

（例：投資、副業、不動産など）

「うまくいかなかった理由は、何だと思いますか？」

（ここで共感を示す）
「分かります。○○さんのお気持ち、よく分かります。」
```

#### 3. 目標を具体化する

```
「もし印税収入が入るようになったら、
どのくらいを目指したいですか？」

（答えを待つ。もし曖昧なら）

「月5万円くらいですか？10万円くらいですか？それ以上ですか？」

「その収入があったら、生活はどう変わりそうですか？」

（具体的なイメージを引き出す）
```

#### 4. 時間を確認する

```
「週にどのくらいの時間を使えそうですか？」

「本業との両立は問題なさそうですか？」

「なるほど。それなら、無理なく続けられそうですね。」
```

### ヒアリングのポイント

- **8割聞く、2割話す**（聞き役に徹する）
- **共感を示す**（「なるほど」「分かります」を多用）
- **メモを取る**（シミュレーションで使う）
- **遮らない**（顧客の話を最後まで聞く）
- **オープンクエスチョン**（「はい/いいえ」で答えられない質問）

### 顧客の反応に応じた対応

#### 話しやすい場合
→ そのまま聞き続ける。深掘りする。

#### 話しにくそうな場合
→ 「大丈夫です。リラックスしてくださいね」と声をかける。

#### 警戒心が強い場合
→ 「今日は相談だけでも大丈夫です。無理に何かを決める必要はありません」と伝える。

```

今日は、○○さんの目標に合わせてお話ししますね。」
```

```
「先ほどお聞きした目標は、月○○万円の印税収入でしたね。

```

**顧客の反応に応じた対応：**
- 「はい、そうです」→ そのまま進む
- 「実はもう少し高めに考えています」→ 「なるほど。それなら、より具体的にシミュレーションできますね」

### スライド3：他の投資との比較

```
「まず、他の投資と比較してみましょう。
月20万円の収入を得るには、どのくらい必要か見てみますね。」

（スライドを表示しながら）

「不動産投資だと3,000万円以上。
高配当株だと4,800万円。
インデックス投資だと6,000万円。

一方、Kindle印税は...後でシミュレーションしますね。」

（ここで顧客の反応を見る）
```

**顧客の反応に応じた対応：**

#### 「不動産投資を考えていた」
```
「なるほど。不動産投資も良い選択肢ですよね。
でも、初期費用が高いのがネックですよね。
○○さんは、どのくらいの資金を考えていましたか？」
```

#### 「株をやっている」
```
「なるほど。株も良い投資ですよね。
でも、株価の変動が気になりますよね。
○○さんは、どんな銘柄を選んでいますか？」
```

#### 「投資は初めて」
```
「なるほど。投資は初めてなんですね。
でも、資産形成に興味があるということですよね。
○○さんは、どんな収入源を考えていましたか？」
```

**ポイント：**
- **押し付けない**（「Kindle印税が一番良い」とは言わない）
- **顧客の意見を尊重する**（「なるほど」と共感）
- **比較検討を促す**（「どちらが良いか、一緒に考えましょう」）

### スライド7：1冊あたりの平均印税

```
「1冊あたりの平均印税は、月5,000円です。

○○さんは、この数字、どう思われますか？」
```

**顧客の反応に応じた対応：**

#### 「5,000円は少ない」
```
「なるほど。確かに、1冊だけだと5,000円ですよね。
でも、10冊で5万円、20冊で10万円、50冊で25万円...
積み上げれば、増えていきますよね。

○○さんは、何冊くらいを目標に考えていますか？」
```

#### 「5,000円で良い」
```
「そうですね。1冊5,000円でも、10冊で5万円になりますよね。
○○さんは、どのくらいの冊数を目標に考えていますか？」
```

#### 「本当に5,000円入るのか？」
```
「なるほど。その疑問、とても重要です。
実績数字でお答えしますね。」

（スライド7.5に進む）
```

### スライド7.5：実績・証拠

```
「本当に印税は入るのか？実績数字でお答えします。」

（スライドを表示しながら）

「100冊以上出版している方々の実績平均が、月5,000円です。
計算根拠は、Kindle Unlimitedのページ単価、0.4〜0.5円です。

例えば、100ページの本が月100回読まれた場合：
100ページ × 0.45円 × 100回 = 約4,500円

これが、1冊5,000円という数字の根拠です。

○○さんは、この根拠、どう思われますか？」
```

**ポイント：**
- **根拠を明確に示す**（数字で説明）
- **顧客に判断させる**（「どう思われますか？」と聞く）
- **押し付けない**（「絶対に入ります」とは言わない）

## 🎯 フェーズ5：スライド8（20〜25分）

### スライド8：シミュレーション

```
「では、○○さん専用のシミュレーションを作成しましょう。
目標や使える時間を入力して、必要な冊数と期間を確認しますね。」

（シミュレーターを開く）

「○○さんの目標は、月○○万円でしたね。
これを入力しますね。」

（数値を入力しながら）

「週に使える時間は、○○時間でしたね。
これも入力しますね。」

（結果を表示）

「○○さんの場合、約○冊で目標達成できます。
1冊90分で作成可能なので、週○冊ペースなら約○ヶ月で目標達成ですね。

○○さんは、この数字、どう思われますか？」
```

**顧客の反応に応じた対応：**

#### 「思ったより少ない」
```
「そうですね。意外と少ないですよね。
でも、これが現実的な数字です。
○○さんは、このペースなら続けられそうですか？」
```

#### 「思ったより多い」
```
「なるほど。確かに、○冊は多いかもしれませんね。
でも、1冊90分で作れるので、週末の数時間で進められます。
○○さんは、このペースなら続けられそうですか？」
```

#### 「本当に90分でできるのか？」
```
「なるほど。その疑問、とても重要です。
後で詳しく説明しますね。
まず、このシミュレーションの数字、どう思われますか？」
```

**ポイント：**
- **顧客と一緒に入力する**（一人で進めない）
- **顧客の反応を見る**（表情や声のトーンを確認）
- **疑問を解消する**（不安を残さない）

## 🎯 まとめ

### 成功のポイント
1. **聞く：話す = 8：2**（顧客の話を8割聞く）
2. **押し付けない**（「買ってください」は絶対に言わない）
3. **共感を最優先**（「なるほど」「分かります」を多用）
4. **顧客が決断する**（自分から「これが欲しい」と言わせる）
5. **疑問をすべて解消**（不安を残さない）

### 失敗しないために
- ❌ 「絶対に稼げます」と言わない
- ❌ 「今だけ特別価格です」と言わない
- ❌ 「買ってください」と言わない
- ❌ 顧客の話を遮らない
- ❌ 強引に押し込まない

このスクリプトを参考に、自然な会話を心がけてください。
顧客が「自分が求めていたものだ」と感じられるように。
